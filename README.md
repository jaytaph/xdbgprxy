# xdbgprxy



```
Usage of xdbgprxy:
      --ide-host string      IP to your IDE (default "127.0.0.1")
      --ide-port int         Port to your IDE (default 9003)
      --listen-host string   IP to listen on (default "127.0.0.1")
      --listen-port int      Port to listen on (default 9003)
      --no-color             No ANSI color output
      --verbose              Verbose output (default true)

```
